Elementos TABLE e DIV

Vamos aprender como trabalhar com DIVs e tabelas na linguagem HTML 5 e CSS inline

TABLE

Tabela da maneira antiga. Não possui separação entre conteúdo e cabeçalho, apenas linhas e celular.

Lembre-se de nunca usar tabelas para organizar layouts de páginas. Antigamente, usuávamos, Hoje, não. Tabelas servem apenas para MOSTRAR DADOS TABULADOS (como se fosse uma planilha)

RA Nome
123456 José Neto
789456 Raquel Oliveira

Tabelas modernas para apresentar dados usam thead e tbody

RA: Nome:
1234 José Neto
5678 Raquel Oliveira

DIVs

A tag DIV como o próprio nome diz, é divisão (espacial). Usamos DIVs para organizar espacialmente elementos na página. Atualmente não usamos tanto DIV para criar "seções" como rodapé, conteúdo e cabeçalho (hoje usamos a tag section). No entanto é perfeitamente possível fazer só com DIVs.
